# Zonaprop Scraper & Web App Monorepo

This repository is split into two primary components to form a modern local tool that allows you to scrape property listings from Zonaprop and visualize them beautifully on a glassmorphism-styled map.

## Project Structure

- **`/scraper`**: A Python-based scraper powered by Cloudscraper, BeautifulSoup, and precise JSON structure parsing. It safely bypasses standard bot protections and extracts reliable map coordinate data and price information.
- **`/web-app`**: A modern frontend application built with Vite, React, TypeScript, and Leaflet. It consumes the `.csv` outputs generated by the scraper to display properties on an interactive map.

## 1. Running the Scraper

To fetch the latest data for any Zonaprop search URL:

```bash
cd scraper
uv sync # To install dependencies via UV (or use standard standard pip installation)
.venv/bin/python main.py "https://www.zonaprop.com.ar/casas-alquiler-cordoba-2-habitaciones.html"
```

The script will:
1. Bypass bot protection and fetch the JSON state for all listings.
2. Produce a `.csv` database inside the `/scraper/data/` folder.
3. Automatically copy the latest `.csv` output to the `/web-app/public/data.csv` folder so the UI always has the most up to date info.

## 2. Viewing the Map Web App

Once the scraper has generated `data.csv`, you can boot up the interactive dark-mode map frontend:

```bash
cd web-app
npm install 
npm run dev
```

Visit `http://localhost:5173` to interact with the properties. Hovering sidebar cards will dynamically emphasize map markers, and the styling uses a premium transparent look.
